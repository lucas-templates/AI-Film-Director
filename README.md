# AI Film Director - Landing Page

A modern, single-page landing page for an AI-powered film production platform. This demo showcases script analysis and virtual casting tools with an elegant, cinematic design.

AI Film Director is a demonstration landing page that presents an AI-powered film production platform. The site features interactive demos for script analysis and virtual casting tools, designed with a cinematic aesthetic using a dark theme with gold and purple accents. The page is fully responsive and includes smooth animations and modern UI/UX elements.

- **Script Analysis Demo**: Interactive tool that analyzes screenplay scripts and provides insights on:
  - Word count, character count, and line statistics
  - Scene structure analysis
  - Dialogue evaluation
  - Overall script score with AI-powered recommendations
  - Character development suggestions

  - Gradient backgrounds and cinematic color scheme

  - Mobile-responsive navigation menu
  - Section-based navigation

## üöÄ Setup Instructions

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for best experience)

### Installation

1. **Clone or Download** the project files to your local machine

2. **Open the Project**
   - Option 1: Simply open `index.html` in your web browser
   - Option 2: Use a local web server for better performance:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (with http-server)
     npx http-server
     
     # Using PHP
     php -S localhost:8000
     ```
   - Then navigate to `http://localhost:8000` in your browser

3. **No Build Process Required**
   - This is a static website with no dependencies to install
   - All external resources (Bootstrap) are loaded via CDN
   - Ready to use immediately after download

### Customization

- **Colors**: Modify the color scheme in `styles.css` (search for `#FFD700`, `#7B1FA2`, `#1976D2`)
- **Content**: Edit text content directly in `index.html`
- **Functionality**: Enhance features in `script.js`
